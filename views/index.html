About :

Closeable Detectors:

Simple Detector - Learns the most correlated column to each column and calculates their regression line.
then, detect samples in the train csv file that are distant from this line and present them as anomalies.

Hybrid Detector - Same as the Simple one but also contains a minimum circle algorithm.
It Calculates the minimum circle to contain all the samples in the learn csv file, and check which sample in
the test csv file does not fit in this circle, and presents them as anomalies as well.
